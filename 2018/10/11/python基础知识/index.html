Python基础知识

数据类型和变量比较简单，就可以不用复习了。直接看看list和tuple

list

列表，python内置数据类型。

['张三', '李四', '钱五']

'张三'

索取最后一个元素

'钱五'

依次类推，获取倒数第2个，第3个

'李四'

'张三'

list是一个可变的列表，可以追加元素。

['张三', '李四', '钱五', '赵六']

也可以插入指定位置，使用insert(index , ‘’)

['小明', '张三', '李四', '钱五', '赵六']

删除元素: pop(index)

['张三', '李四', '钱五', '赵六']

['张三', '李四', '钱五']

可以直接队某个元素进行赋值。

['小明', '李四', '钱五']

list里面的元素的数据类型也可以不同

['name', 18, ['name', 18]]

我们可以看到上面的例子中，列表里可以有列表，我们来拿到这个元素。可以用二维数组的方法。

'name'

tuple

元组，与list的区别在于，tuple一旦初始化就不能修改，而且使用的是圆括号。

('a', 'b')

判断

与其他语言类似，介绍几个注意点

  • elif是else if 的缩写
  • if后面不用括号
  • input输入的数据是字符串，不能直接和int型比较

循环

循环也和其他语言类似，介绍几个注意点

  • for循环指的是for x in … 把每个元素代入x，然后输出
  • while循环后面也没有括号
  • 不要滥用break和continue，这会使得代码变得复杂

dict

字典，内置数据类型。

其实就是js里面的对象，一般也成为map，使用键-值（key-value）存储，具有极快的查找速度。原理和我们平时查字典是一样的！

dict使用{}大括号

{'name': '小明', 'age': 18}

'小明'

18

为了避免key不存在发生的报错，我们可以检查key是否存在

False

'None'

删除key

{'age': 18}

与list比较，dict实际上是利用空间来换取时间的一种方法

key必须是不可变对象！牢记

这是因为dict根据key来计算value的存储位置，如果每次计算相同的key得出的结果不同，那dict内部就完全混乱了。这个通过key计算位置的算法称为哈希算法（Hash）。

要保证hash的正确性，作为key的对象就不能变。在Python中，字符串、整数等都是不可变的，因此，可以放心地作为key。而list是可变的，就不能作为key：

set

set仅是一组存储key的集合，不存储value

  • 通过add(key)可以添加key
  • 通过remove(key)可以移除key
  • set不会存储重复的key，如果有重复，set将会删除重复的元素。
  • 最常用的key是字符串
['key1', 'key2', 'key3']

返回多个值

(1.2246467991473532e-16, 0.8660254037844387)

哦，原来返回多个值的话，其实是返回来了一个tuple，返回一个tuple可以省略括号，而多个变量可以同时接受一个tuple，按位置赋给对应的值。

默认参数

给参数赋一个具体值，这样在调用的时候就可以不给该参数赋值了，如果想改变这个值，也可以直接赋值

但要记住默认参数一定要在最后面

(1.2246467991473532e-16, 0.8660254037844387, 3.141592653589793)

定义默认参数要记住，默认参数必须指向不变对象！！
